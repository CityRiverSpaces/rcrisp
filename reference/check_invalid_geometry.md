# Check and fix invalid geometries

Check and fix invalid geometries

## Usage

``` r
check_invalid_geometry(sf_obj)
```

## Arguments

- sf_obj:

  An object of class
  [`sf::sf`](https://r-spatial.github.io/sf/reference/sf.html) or
  [`sf::sfc`](https://r-spatial.github.io/sf/reference/sfc.html)

## Value

[`sf::sf`](https://r-spatial.github.io/sf/reference/sf.html) or
[`sf::sfc`](https://r-spatial.github.io/sf/reference/sfc.html) object
with valid geometries
