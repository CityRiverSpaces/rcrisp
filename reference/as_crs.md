# Standardise the coordinate reference system (CRS) of an object

Standardise the coordinate reference system (CRS) of an object

## Usage

``` r
as_crs(x, allow_geographic = FALSE)
```

## Arguments

- x:

  An object of class `sf`, `sfc`, `bbox`, or a numeric or character
  vector representing a CRS (e.g., EPSG code). If `numeric`, the value
  should be an unrestricted positive number representing a valid EPSG
  code.

- allow_geographic:

  Logical, whether to allow geographic CRS (lat/lon).

## Value

An object of class
[`sf::crs`](https://r-spatial.github.io/sf/reference/coerce-methods.html)
with a valid CRS.
