# Get the bounding box of a city

Get the bounding box of a city

## Usage

``` r
get_osm_bb(city_name)
```

## Arguments

- city_name:

  A character vector of length one

## Value

A `bbox` object with the bounding box of the city

## Examples

``` r
if (FALSE) { # interactive()
get_osm_bb(city_name = "Bucharest")
}
```
