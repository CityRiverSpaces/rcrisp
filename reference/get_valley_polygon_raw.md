# Create vector/polygon representation of valley raster mask

Create vector/polygon representation of valley raster mask

## Usage

``` r
get_valley_polygon_raw(valley_mask)
```

## Arguments

- valley_mask:

  raster mask of valley pixels

## Value

polygon representation of valley area as object of class
[`sf::sfc`](https://r-spatial.github.io/sf/reference/sfc.html)
